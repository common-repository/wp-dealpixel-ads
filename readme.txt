WP DealPixel Ads - allows you to list DealPixel.com deals in your sidebar and make money with the DealPixel affiliate network.

== Description ==

Allows you to insert the latest deals from dealpixel.com in your sidebar.
Easily earn money by joining the DealPixel affiliate network and driving traffic to dealpixel.com! Make 25% off of each sale. A lot better than amazon, ebay, google ads, or most other affiliate networks. Find out more [here](http://dealpixel.com/affiliate-program/).

A few of the features:

*   Choose how many deals to show
*   Hides the affiliate link

More about Dealpixel.com:

Dealpixel.com is a daily deal site for creative and web professionals. They offer huge discounts on things like icons, wordpress themes, stock images, online courses, and more. They also have a very generous affiliate program where they pay 25%.

== Installation ==

WP DealPixel Ads is installed like any other Wordpress plugin.

1) Download the latest package from the downloads page (http://wordpress.org/extend/plugins/wp-dealpixel-ads/).

2) Upload the whole directory into your wordpress plugins directory (usually, yoursite/wp-content/plugins). The plugin must reside in a sub-folder called "wp-dealpixel-ads" (lower-case!) as supplied in the zip file, else it WON’T WORK

3) Go into the Wordpress admin panel, find Plugins and WP DealPixel Ads should be listed. Click Activate to activate!


Using WP DealPixel Ads 

1) Go to your theme widgets and find the WP DealPixel Ads widget and place it.

2) Configure the settings in the options panel.
